# Educational Keylogger Project

## Overview
This project is a simple keylogger built for educational purposes in cybersecurity learning. It captures keyboard input and stores it in a local log file.

## Features
Captures keystrokes
Stores logs locally
Simple log viewer
Stops logging using ESC key

## Technologies Used
Python
pynput library

## Learning Outcomes
Understanding OS input hooks
Logging system events
Ethical considerations in cybersecurity tools

## Important Note
This software is created only for educational purposes and should not be used for malicious activities.
